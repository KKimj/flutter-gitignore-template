# flutter-template

Flutter & Firebase project template

## [.gitignore](.gitignore)

## Getting started

```
$ git clone https://github.com/KKimj/flutter-template.git
```

### Build flutter project with Chrome 
```
$ flutter run -d chrome
```

### Test .dart /test
```
$ flutter test --machine test/widget_test.dart
# or just
$ flutter test
```
